##Functioning of the model
process<- function(A, K) {
  #Creation
  for(i in 1:K) {
    boundary= c()
    for(element in A) {
      if(! (element +1) %in% A) {
        boundary<- append(boundary, (element+1))
      }
      if(! (element -1) %in% A) {
        boundary<- append(boundary, (element-1))
      }
    }
    A<- append(A, sample(boundary, 1))
  }
  
  #Annihilation
  for(i in 1:(K-1)) {
    length= length(A)
    remove_index= sample(1:length, 1)
    A<- A[-c(remove_index)]
  }
  return(A)
}

1 .

Let the number of elements of \(A_n\) be \(N\) at time \(n\).

At time \(n+1\), \(K\) new sites are added one by one, making the total number of occupied sites \(N+K\). After that \(K-1\) sites are chosen uniformly from those \(N+K\) sites and are deleted, one by one.

Therefore, in the end, the total number of sites result to \(N+K-(K-1)= N+1\).

##Functioning of L(A_n) and G2(A_n)
L<- function(A) {
  A_sorted= sort(A)
  A_gaplength= diff(A_sorted)-1
  sum(A_gaplength)
}

G2<- function(A) {
  A_sorted= sort(A)
  A_gaplength= diff(A_sorted)-1
  sum(A_gaplength >=2)
}
##Finding L_n and G2_n for n=1000 over 100 trials
L_tn= matrix(nrow= 100, ncol= 1000)
G2_tn= matrix(nrow= 100, ncol= 1000)
trials= 1:100; n_values= 1:1000
for(t in trials) {
  A= c(0); K= 3
  for(n in n_values) {
    A= process(A, K)
    L_tn[t,n]= L(A)
    G2_tn[t,n]= G2(A)

}

From theorem \(2.1.1.\) (Dorothea Maria Eberz-Wagner 1999), we say \(\lim sup_{n\to\infty} \frac{L(A_n)}{h(n)}= \frac{K-1}{K}\), which means \(\exists\; N_0 \in \mathbb{N}\) s.t. \(L(A_n)\) and thus \(\overline{L(A_n)}= L_n\leq \frac{2}{3}h(n)\; \forall\; n\geq N_0\) as, in this case \(K=3\).

We try to plot the line graph of \(\frac{L(A_n)}{h(n)}\), and also show the line \(y=\frac{2}{3}\), under which the graph should lies \(\forall\; n\geq N_0\).

We try to show, as Theorem \(2.1.2\) (Dorothea Maria Eberz-Wagner 1999), that \[ \mathbb{P}\left[G^{(2)}(A_n)= 1\; \text{ i.o.}\right]= 1 \] \[ \implies \mathbb{P}\left[\;\forall N\geq 1\; \exists n\geq N\; s.t.\; G^{(2)}(A_n)= 1\right]= 1 \] \[ \implies \mathbb{P}\left[\;\forall N\geq 1\; \exists n\geq N\; s.t.\; \overline{G^{(2)}(A_n)}= G^{(2)}_n(A_n)= 1\right]= 1 \]

Trying to find the empirical probability through statistics….

##Showing by plot P[G2_n(A_n)=1 i.o.]= 1

pG2_1= matrix(nrow= 100, ncol= 1000)
for(t in trials) {
  max_g2n_1= 1
  for(n in n_values) {
    if(G2_tn[t, n]== 1) {
      max_g2n_1= n
    }
  }
  for(n in n_values) {
    if(n<= max_g2n_1) {
      pG2_1[t,n]= 1
    } else {
      pG2_1[t,n]= 0
    }
  }
}
pG2n_1= c()
for(n in n_values) {
  pG2n_1[n]= mean(pG2_1[,n])
}
#print(pG2n_1)
plot(n_val, c(1, pG2n_1),
     xlab= "n Values", ylab= "P[N: N>=n, G2(N)=1]",
     main= "Probability of Existence of Gap-2", type = "l")

We see for lower values of \(n\), \(\mathbb{P}= 1\). But as we go higher, the probability decreases since there are limited number of samples taken into consideration.

Therefore, if we take the whole Natural Number line, we can say \(\mathbb{P}\left[G^{(2)}(A_n)= 1\; \text{ i.o.}\right]= 1\)
